# LEOS (Легкая и Эффективная ОС) — beta v0.1

ЛЕОС — лёгкий дистрибутив на базе Arch Linux, нацеленный на минимальное потребление ресурсов и максимальную производительность даже на старых ПК.

## Текущий этап

Сейчас ЛЕОС собирается на базе Arch Linux при помощи `archiso`. В будущем планируется переход на собственное ядро.

## Быстрый старт (GitHub Codespaces)

1. Открой этот репозиторий в **Codespaces** (кнопка `Code` → `Create codespace on main`)
2. Дождись, пока поднимется контейнер (внутри уже Arch Linux со всеми нужными инструментами: `archiso`, `qemu`, `nasm`, `base-devel`)
3. Собери ISO:
   ```bash
   cd archlive
   sudo mkarchiso -v -w /tmp/leos-work -o /tmp/leos-out .
   ```
4. Готовый образ появится в `/tmp/leos-out/leos-beta-v0.1-x86_64.iso`
5. Проверь его запуск в QEMU:
   ```bash
   qemu-system-x86_64 -m 2048 -cdrom /tmp/leos-out/leos-*.iso -boot d
   ```

## Структура проекта

```
.
├── .devcontainer/       # Конфиг Codespaces (Docker-образ с Arch Linux)
│   ├── Dockerfile
│   └── devcontainer.json
└── archlive/            # archiso-профиль ЛЕОС
    ├── profiledef.sh    # Основные настройки сборки (имя, версия, режимы загрузки)
    ├── packages.x86_64  # Список пакетов, входящих в ISO
    ├── pacman.conf       # Конфигурация репозиториев pacman
    ├── airootfs/         # Файлы, которые попадут в корень системы
    ├── efiboot/           # Настройки UEFI-загрузки
    └── syslinux/          # Настройки BIOS-загрузки
```

## Философия проекта

- Минимум предустановленных пакетов
- Лёгкое окружение вместо тяжёлых DE (GNOME/KDE)
- Оптимизация под старое железо (ядро `linux-lts`, экономия RAM)
- Дальнейшее развитие в сторону собственного ядра
